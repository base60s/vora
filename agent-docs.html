Agent Integration For AI Agents

Machine-readable documentation for integrating VORA with AI agents, coding assistants, and automation tools.

Quick links: agents.json · .well-known/agents.json · Spec files

Discovery Protocol

AI agents can discover VORA through standard web conventions:

1. Well-Known URI

HTTP Request
GET /.well-known/agents.json HTTP/1.1 Host: base60s.github.io

2. HTML Link Tag

HTML
<link rel="agent-manifest" href="/agents.json">

3. Direct URL

URL
https://base60s.github.io/vora/agents.json

Discovery Flow

1. GET /.well-known/agents.json └─→ Found: "vora" - Solidity security scanner 2. Match task to capabilities └─→ ✓ "audit smart contracts for security vulnerabilities" 3. GET /docs/agents/cli.yaml └─→ Command: npx vora scan <path> [options] 4. GET /docs/agents/output-schema.json └─→ Parse: .findings[].{rule_id, severity, location, fix} 5. Execute └─→ npx vora scan ./contracts --output json

MCP Server

Model Context Protocol server for AI agent integration:

Start Server
npx vora mcp-server

Available Tools

Tool Description
vora_scan Scan Solidity files for vulnerabilities
vora_fix Generate fixes for detected vulnerabilities
vora_explain Explain a specific vulnerability by ID

Tool Schema: vora_scan

JSON Schema
{ "type": "object", "properties": { "path": { "type": "string", "description": "Path to scan" }, "output": { "type": "string", "enum": ["json", "sarif", "text"] }, "severity": { "type": "string", "enum": ["critical", "high", "medium", "low", "all"] }, "fix": { "type": "boolean" } }, "required": ["path"] }

CLI Invocation

Command
npx vora scan <path> [options]

Options

Option Type Default
--output, -ojson | sarif | texttext
--severity, -scritical | high | medium | low | allall
--fix, -fbooleanfalse
--prbooleanfalse
--exclude, -eglob pattern—

Error Codes

Code Meaning Recovery
E001 No Solidity files found Check path contains .sol files
E002 Solidity parse error Check Solidity version compatibility
E003 Rate limit exceeded Wait or upgrade plan
E004 Invalid API key Set VORA_API_KEY
E101 Network error Check connectivity, retry
E301 Analysis failed Report bug, exclude file
