How It Works

//01

Parse & Build AST

Solidity files are parsed into an abstract syntax tree. Control flow and data flow graphs are constructed for each function.

//02

Pattern Matching

Static analysis runs 200+ detection rules. ML model identifies complex vulnerabilities from learned audit patterns.

//03

Generate Fixes

For each vulnerability, a fix is generated using code templates and verified against the original bug pattern.

//04

Submit PR

Fixes are committed to a new branch and a pull request is opened with detailed vulnerability report.

Example: Reentrancy Fix

VORA detects the vulnerability pattern and generates a fix using checks-effects-interactions.

Vault.sol Vulnerable
140function withdraw(uint amount) external {
141 require(balances[msg.sender] >= amount);
142 // External call before state update
143 (bool ok,) = msg.sender.call{value: amount}("");
144 require(ok);
145 balances[msg.sender] -= amount;
146}
